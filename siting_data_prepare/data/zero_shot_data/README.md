# Zero-shot 参考音频（网盘下载）

本目录存放 **127 条精选参考音频**（14 角色），是 CosyVoice Zero-shot Speaker 注册的声纹来源，全部人工从《甄嬛传》原声中挑选。

## ⚠️ 音频文件单独放网盘

由于体积较大，WAV 文件未随代码仓库分发，请从网盘下载：

- **网盘链接**：https://pan.baidu.com/s/1_uacMyl9y5YvrWsh77Mtww?pwd=kcs7
- **提取码**：kcs7
- **网盘目录**：`zero_shot_data/`（14 个角色子目录，共 127 段 WAV）

## 使用方式

1. 下载网盘压缩包并解压
2. 将 14 个角色子目录及其中的 `.wav` 文件放入本目录
3. 解压后目录结构应为：

```
zero_shot_data/
├── 甄嬛/                      ← 每个角色一个子目录
│   ├── zhenhuan_xxxx.wav
│   └── ...
├── 华妃·年世兰/
├── 皇上·爱新觉罗·胤禎/
├── 乌拉那拉·宜修(皇后)/
├── ...（共 14 个角色目录）
```

4. 运行 `../../siting_cosyvoice_tts/scripts/register_speakers.py` 进行 Zero-shot Speaker 注册
